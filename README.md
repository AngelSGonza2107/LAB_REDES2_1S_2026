## Descripción del Curso

El curso **Redes de Computadoras 2** se enfoca en el **diseño, implementación y análisis de redes LAN y WLAN**, integrando conceptos fundamentales y aplicados para la construcción de infraestructuras de red modernas.

Durante el laboratorio se trabajan temas como:

- Topologías de red
- Protocolos de enrutamiento dinámico
- Segmentación mediante VLAN
- Asignación dinámica de direcciones con DHCP
- Control de tráfico usando ACLs
- Implementación de servicios de red como DNS, HTTP y HSRP
